# 运行说明

## 环境要求

- Python: `D:/software/anaconda3/python.exe`（3.12.13；若不可用可改用本机等价 3.12 解释器）
- 依赖安装：

```powershell
python -m pip install -r requirements.txt
```

## 运行测试

```powershell
python -m pytest -q
```

## 冒烟运行

```powershell
python main.py --seed 42
```

运行后会在 `output/` 目录生成：

- `latest_schedule.json`：包含 `scheduled_items`、`unscheduled_tasks`、`objective_value`、`constraint_stats`
- `cycle_log.jsonl`：每行一个周期日志，含周期状态快照、已选任务、未规划任务、约束违规统计与目标值

## 配置说明

主配置文件：`config/planner_config.json`

- `runtime`：时间轴、求解超时、随机种子
- `simulation`：任务数量、DAG 组数、关键任务名
- `constraints`：资源容量（含并发核心阈值）、姿态角转动时间系数、热电限制等
- `objective_weights`：收益与惩罚权重
- `logging`：输出目录与文件名
