# 配置文件注释说明

本文用于说明当前静态基线规划所使用的 JSON 配置字段，字段口径以代码实现为准。

## runtime.json

| 字段 | 类型 | 单位 | 必填 | 默认值 | 作用 | 代码引用 |
|---|---|---|---|---|---|---|
| input_mode | string | - | 否 | static | 当前阶段仅支持 static | src/scheduler/config.py |
| data_dir | string | 路径 | 否 | data | 输入数据根目录 | src/scheduler/config.py |
| tasks_file | string | 文件名 | 否 | latest_small_tasks_pool.json | 任务池文件名 | src/scheduler/config.py |
| windows_file | string | 文件名 | 否 | latest_windows.json | 可见窗口文件名 | src/scheduler/config.py |
| seed | int | - | 否 | 666 | 随机种子 | src/scheduler/pipeline.py |
| time_horizon | int | 秒 | 是 | 240 | 求解时域上界 | src/scheduler/pipeline.py |
| solver_timeout_sec | number | 秒 | 是 | 30 | CP-SAT 限时 | src/scheduler/pipeline.py |
| initial_attitude_angle_deg | number | 度 | 是 | 无 | 初始姿态角 | src/scheduler/pipeline.py |
| thermal_time_step | number | 秒 | 否 | time_step | 热模型离散步长 | src/scheduler/pipeline.py |
| initial_temperature_fallback | number | 摄氏度 | 否 | 25.0 | 上轮状态缺失时回退温度 | src/scheduler/pipeline.py |
| thermal_initial_source | string | - | 否 | last_state_first | 初始温度来源策略 | src/scheduler/pipeline.py |
| replan_state_max_age_sec | number | 秒 | 否 | 600 | 历史状态最大可用年龄 | src/scheduler/pipeline.py |
| heuristic_log_every_n | int | 次 | 否 | solver_progress_every_n_solutions | 启发式日志频率 | src/scheduler/config.py |
| cpsat_log_every_n | int | 次 | 否 | solver_progress_every_n_solutions | CP-SAT 日志频率 | src/scheduler/config.py |
| solver_progress_every_n_solutions | int | 个解 | 否 | 10 | 中间解日志采样频率 | src/scheduler/config.py |

## constraints.json

| 字段 | 类型 | 单位 | 必填 | 默认值 | 作用 | 代码引用 |
|---|---|---|---|---|---|---|
| cpu_capacity | number | 核 | 是 | 无 | CPU 容量上限 | src/scheduler/problem_builder.py |
| gpu_capacity | number | 卡 | 是 | 无 | GPU 容量上限 | src/scheduler/problem_builder.py |
| memory_capacity | number | 单位值 | 是 | 无 | 内存容量上限 | src/scheduler/problem_builder.py |
| power_capacity | number | 功耗单位 | 是 | 无 | 功率容量上限 | src/scheduler/problem_builder.py |
| attitude_time_per_degree | number | 秒/度 | 是 | 无 | 姿态切换时间系数 | src/scheduler/problem_builder.py |
| thermal.warning_threshold | number | 摄氏度 | 是 | danger_threshold-10 | 热预警阈值 | src/scheduler/config.py |
| thermal.danger_threshold | number | 摄氏度 | 否 | 100.0 | 热危险阈值 | src/scheduler/config.py |
| thermal.max_warning_duration | number | 秒 | 否 | 60 | 连续预警可容忍上限 | src/scheduler/config.py |
| thermal.warning_thermal_load | number | 负载单位 | 否 | 无 | 高热任务判定阈值 | src/scheduler/cpsat_improver.py |
| thermal.thermal_concurrency_limit | int | 个 | 否 | 1 | 高热任务并发上限 | src/scheduler/cpsat_improver.py |
| thermal.power_safe_ratio | number | 比例 | 否 | 0.7 | 功率安全阈值比例 | src/scheduler/cpsat_improver.py |
| thermal.env_temperature | number | 摄氏度 | 否 | 20.0 | 环境温度 | src/scheduler/thermal_model.py |
| thermal.coefficients.a_p | number | - | 否 | 0.002 | 功耗发热系数 | src/scheduler/thermal_model.py |
| thermal.coefficients.a_c | number | - | 否 | 0.03 | 并发线性发热系数 | src/scheduler/thermal_model.py |
| thermal.coefficients.lambda_concurrency | number | - | 否 | 0.01 | 并发二次发热系数 | src/scheduler/thermal_model.py |
| thermal.coefficients.k_cool | number | - | 否 | 0.005 | 线性散热系数 | src/scheduler/thermal_model.py |
| objective_scaling.* | [min,max] | - | 否 | 见配置默认 | 目标分量归一化区间 | src/scheduler/objective_engine.py |

## logging.json

| 字段 | 类型 | 单位 | 必填 | 默认值 | 作用 | 代码引用 |
|---|---|---|---|---|---|---|
| output_dir | string | 路径 | 否 | output | 输出目录（可被 CLI 参数覆盖） | main.py |
| schedule_file | string | 文件名 | 否 | latest_schedule.json | 计划结果文件名（历史兼容） | main.py |
| cycle_log_file | string | 文件名 | 否 | cycle_log.jsonl | 周期日志文件名（历史兼容） | main.py |
| task_pool_file | string | 文件名 | 否 | latest_task_pool.json | 任务池快照文件名（历史兼容） | main.py |
| visibility_windows_file | string | 文件名 | 否 | latest_visibility_windows.json | 窗口快照文件名（历史兼容） | main.py |
